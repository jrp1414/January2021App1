String Interpolation Examples

Things we can do using String Interpolation

  • Simple Code Expression : 4+5 {{4+5}}
  • Property Example 1 : {{num1}}
  • Property Example 2 : {{num1 + num2}}
  • Property Example 3 : {{termsAndConditions}}
  • Property Example 4 : {{name}}
  • Property Example 4 : {{firstName + " " + lastName}}
  • Object Example 1: {{person}}
  • Object Example 2: {{person?.Name}}
  • Method Example 1: {{getName()}}
  • Object Example 4: {{person?.Address?.AddLine1}}

Things we cannot do using String Interpolation

  • Cannot Create new objects inside string Interpolation Syntax
  • Cannot write chained expressions
  • Cannot write comma seprate expressions
